    - Cómo instalar o ejecutar el proyecto.
      Requisitos.
        - JDK 21.
        - Apache NeatBeans desde version 26.
        - Xampp
          Para el servidor de:
            - Tomcat - Credenciales: root/root
            - Apache + Mysql - Credenciales: root/""
        Instrucciones:
          1. Clonar el repositorio.
          2.Abrir como proyecto web en Apache NetBeans
          3.Configurar servidor Tomcat con las credenciales descritas.
          4.Importar el backup de la base de datos que esta en el repositorio mismo. (Ya tiene credenciales creadas - admin/admin, root/root)
          5.Si al abrir el proyecto muestran errores de localizacion de archivos, los mismos estaran en la siguiente ruta del proyecto:
            proyecto_tp_sistema_de_ventas\src\java\librerias
          6.Tener levantado tomcat+MySQL
          7.Correr el sistema.
